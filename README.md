# cxmprs
Rust Program that uses Huffman Coding to compress files.

## Compression
Using:
```bash
# Will output to your_file.cxmp
./cxmprs your_file 
```
#### Try it
There's already built file in `/build`, so you can use it.
```bash
# Output will go to ./file.cxmp
./build/cxmprs ./file.txt
```
